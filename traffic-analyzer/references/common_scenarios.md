# 常见分析场景和示例

## 场景 1: 寻找 AI Tool 创业方向

**用户目标:** 想做一个 AI 工具产品，不知道做什么方向

**分析步骤:**
1. 运行 AI 工具分析，查看当前热门的 AI 产品
2. 筛选增长率 > 50% 的 AI 工具，找到快速增长的赛道
3. 选择流量在 10万-50万 的工具，这个区间竞争相对较小
4. 分析这些工具的类型（代码、设计、音频、视频等）
5. 结合自己的技术能力选择方向

**预期产出:**
- AI 工具流量排行榜
- 高增长 AI 工具列表
- 细分赛道机会分析

## 场景 2: 竞品分析

**用户目标:** 已经有产品方向，想了解竞争对手情况

**分析步骤:**
1. 识别同类竞品（通过关键词或产品名）
2. 对比各竞品的流量规模
3. 分析增长趋势（谁在增长，谁在下降）
4. 查看流量来源类型（学习获客渠道）
5. 评估市场份额分布

**预期产出:**
- 竞品流量对比表
- 增长趋势图
- 市场份额分析

## 场景 3: 市场规模评估

**用户目标:** 评估某个细分市场的规模和潜力

**分析步骤:**
1. 识别该细分市场的所有产品
2. 汇总总流量规模
3. 计算市场增长率
4. 分析市场集中度（是否被头部垄断）
5. 识别增长空间

**预期产出:**
- 市场总规模估算
- 市场增长趋势
- 竞争格局分析

## 场景 4: 发现新趋势

**用户目标:** 发现正在萌芽的新机会

**分析步骤:**
1. 筛选流量 < 50万 但增长率 > 100% 的来源
2. 分析这些高增长来源的共同特征
3. 识别新出现的产品类别
4. 评估趋势的可持续性

**预期产出:**
- 高增长新产品列表
- 新兴趋势总结
- 机会评估

## 场景 5: 流量来源策略分析

**用户目标:** 学习成功产品的获客策略

**分析步骤:**
1. 选择目标产品
2. 分析其流量来源分布（direct/referral/search/social）
3. 识别主要引荐来源
4. 评估不同渠道的表现

**预期产出:**
- 流量来源饼图
- 主要引荐网站列表
- 获客渠道建议

## 实用技巧

### 快速筛选
- **找爆款:** `traffic_diff > 2` (增长超过 200%)
- **找稳定增长:** `traffic_diff > 0.2 and traffic_diff < 1` (增长 20%-100%)
- **找衰退:** `traffic_diff < -0.2` (下降超过 20%)
- **找中型市场:** `traffic > 100000 and traffic < 1000000`

### 组合分析
1. **高增长 + 中等流量** = 最佳机会窗口
2. **高流量 + 低增长** = 成熟市场，难以进入
3. **低流量 + 高增长** = 早期市场，风险较大
4. **流量下降** = 避免进入的赛道

### 关注指标
- **增长率**: 最重要的指标，反映市场热度
- **绝对流量**: 反映市场规模
- **流量占比**: 反映市场地位
- **流量来源**: 反映获客能力

## 常见陷阱

### 误区 1: 只看流量规模
流量大不一定代表好机会，可能竞争已经很激烈。

### 误区 2: 忽视增长趋势
静态流量数据没有意义，要看趋势和变化。

### 误区 3: 不考虑自身能力
即使是好机会，也要评估自己能否做出来。

### 误区 4: 只看短期数据
需要结合多个时间点的数据看长期趋势。

### 误区 5: 忽视数据来源
这是 Stripe 支付页面流量，不是产品总流量。
