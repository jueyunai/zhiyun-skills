# 流量数据分析方法论

## 数据字段说明

### 核心字段
- `type`: 流量类型
  - `direct`: 直接访问
  - `referral`: 引荐流量（从其他网站链接过来）
  - `search`: 搜索引擎流量
  - `social`: 社交媒体流量
  - `ai_assistants`: AI 助手流量
  
- `target`: 流量来源（具体的网站域名）
- `traffic`: 当前流量量
- `prev_traffic`: 之前的流量量
- `traffic_diff`: 流量变化率（-1 到无穷，0 表示无变化）
- `traffic_share`: 该来源占总流量的百分比

## 分析维度

### 1. 增长分析
识别高增长的流量来源，寻找快速增长的市场机会。

**关键指标:**
- `traffic_diff > 100%`: 翻倍增长
- `traffic_diff > 50%`: 快速增长
- `traffic_diff < -20%`: 显著下降

**应用场景:**
- 发现新兴市场和趋势
- 识别衰退的赛道
- 评估市场热度

### 2. 规模分析
按流量规模分层，理解不同规模市场的特征。

**分层标准:**
- 超大流量: > 1,000,000
- 大流量: 500,000 - 1,000,000
- 中等流量: 100,000 - 500,000
- 小流量: 10,000 - 100,000
- 微流量: < 10,000

### 3. 类型分析
按流量类型分类，了解不同获客渠道的表现。

**分析要点:**
- 直接访问通常表示品牌认知度
- 引荐流量表示生态系统健康度
- 搜索流量表示需求强度
- 社交流量表示传播力

### 4. AI 工具专项分析
识别和分析 AI 相关产品的流量表现。

**识别关键词:**
- ai, gpt, claude, openai, anthropic
- midjourney, stable, diffusion
- chatbot, assistant, copilot
- 特定产品名: cursor, lovable, suno, elevenlabs, runway

### 5. 机会发现
寻找"中等流量 + 高增长"的甜蜜点。

**理想特征:**
- 流量: 100,000 - 1,000,000（证明有市场）
- 增长率: > 20%（证明在增长）
- 竞争程度: 适中（有进入空间）

## 商业洞察方法

### 寻找 AI Tool 方向
1. **看增长最快的 AI 工具** → 发现热门赛道
2. **看流量中等但增长快的** → 找到机会窗口
3. **看同类工具的流量对比** → 评估市场容量
4. **看流量来源类型** → 学习获客策略

### 竞品分析
1. 识别同类产品的流量差异
2. 分析增长趋势的不同
3. 对比流量来源渠道
4. 评估市场份额分布

### 市场时机判断
- **进入时机**: 市场有一定规模（10万+流量）且快速增长（50%+）
- **避免时机**: 市场萎缩（负增长）或竞争饱和（流量集中度高）
- **观察信号**: 多个相似工具同时快速增长 = 品类爆发

## 数据局限性

### 注意事项
1. 这是**支付页面**的流量，不是产品首页流量
2. 流量高 ≠ 收入高（需要考虑转化率、客单价）
3. 短期流量波动可能是营销活动导致
4. 某些产品可能使用其他支付方式，未计入 Stripe

### 建议补充数据
- 产品定价信息
- 用户评价和口碑
- 团队背景和融资情况
- 技术难度和壁垒
